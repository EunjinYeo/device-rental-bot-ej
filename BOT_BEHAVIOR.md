# 단말대여봇 동작 정의서

## 대여 흐름

### 1. 단말 목록 조회

**DM으로 요청할 때**
- 사용자가 봇에게 DM으로 `빌려줘` / `대여` / `목록` / `리스트` 중 하나를 포함해 메시지를 보내면
- → 봇이 대여 가능한 단말 목록을 [대여 신청] 버튼과 함께 DM으로 응답

**채널에서 멘션할 때**
- 사용자가 채널에서 `@단말대여봇 목록` 등으로 멘션하면
- → 봇이 단말 목록을 **멘션한 사람에게만 보이는 메시지**(다른 사람 안 보임)로 응답

**그 외 메시지**
- DM으로 다른 말을 보내면 → `안녕하세요! 빌려줘 또는 목록 이라고 보내주시면...` 안내 메시지

---

### 2. 대여 신청

1. 단말 목록에서 [대여 신청] 버튼 클릭
2. 대여 신청 모달 팝업 → 대여 시작일 / 반납 예정일 선택
   - **취소** 클릭 시 → `취소되었습니다. 빌려줘 / 목록 을 다시 입력해주세요.` 메시지
3. [신청하기] 클릭 시
   - 사용자에게 → `대여 신청 완료! {단말명} 대여 가능한지 확인하고 알려드릴게요!`
   - 관리자에게 → 승인 요청 메시지 (신청자 / 단말명 / 자산번호 / 기간 / 📋 시트 링크) + [승인] [기기 변경 후 승인] [거절] 버튼

---

### 3. 관리자 승인/거절

**승인할 때**
- 관리자가 [승인] 클릭
- 사용자에게 →
  ```
  ✅ 대여 완료!
  반납일은 {날짜} 입니다. 반납일을 꼭 지켜주세요.
  - 대여 단말 : {모델명} / {자산번호}
  [반납하기] 버튼
  ```
- 관리자 메시지 → `승인 완료 — {이름} / {단말명} ({기간})`으로 업데이트

**기기 변경 후 승인할 때**
- 관리자가 [기기 변경 후 승인] 클릭 → 대체 기기 선택 모달 팝업
- 선택 후 → 변경된 기기로 승인 처리 (위 승인과 동일)

**거절할 때**
- 관리자가 [거절] 클릭
- 사용자에게 → `{단말명} 대여 신청이 거절됐어요.` + [다른 기기 선택하기] 버튼
- 관리자 메시지 → `거절 — {이름} / {단말명}`으로 업데이트

---

### 4. 반납

1. 사용자가 [반납하기] 버튼 클릭
2. 사용자 메시지 → `@관리자 자리로 가져다주세요! 확인 후 반납 완료 처리됩니다.`로 업데이트
3. 관리자에게 → `반납 확인 요청 / {반납자} / {단말명} / {자산번호}` + [반납 완료] [거절] 버튼

**관리자가 반납 완료 클릭 시**
- 관리자 메시지 → `{단말명} ({자산번호}) 반납이 완료됐어요.`로 업데이트
- 관리자에게 별도 알림 → `{단말명} ({자산번호}) 반납 완료 / 반납자: @{사용자}`

**관리자가 거절 클릭 시**
- 사용자에게 → `{단말명} 반납이 확인되지 않았어요. 관리자에게 문의해주세요.`

---

### 5. 반납일 연장

1. 사용자가 [연장하기] 버튼 클릭 → 연장 신청 모달 팝업 → 새 반납 예정일 선택
2. [신청하기] 클릭 시
   - 사용자에게 → `{단말명} 연장 신청 완료! 관리자 승인 후 확정됩니다.`
   - 원본 알림 메시지 → 버튼 사라지고 `연장 신청이 완료됐어요. 관리자 승인 후 확정됩니다.`로 업데이트
   - 관리자에게 → 연장 승인 요청 (신청자 / 단말명 / 자산번호 / 연장 반납 예정일 / 📋 시트 링크) + [승인] [거절] 버튼

**관리자가 승인 클릭 시**
- 사용자에게 →
  ```
  ✅ 연장 완료!
  반납일이 {날짜} 까지 연장됐어요!
  - 연장 단말 : {모델명} ({자산번호})
  [반납하기] 버튼
  ```
- 관리자 메시지 → `연장 승인 — {이름} / {단말명} → {날짜}`로 업데이트

**관리자가 거절 클릭 시**
- 사용자에게 → `{단말명} 반납일 연장이 거절됐어요. 문의사항은 @관리자에게 연락해주세요.`

---

## 반납 알림 (매일 오전 10:45)

오늘 반납 예정이고 아직 반납 안 된 단말이 있을 때 자동 발송.

**관리자에게**
```
오늘 반납 예정 {N}건
• @{사용자} — {모델명} ({자산번호})
• @{사용자} — {모델명} ({자산번호})
```

**각 대여자에게**
```
오늘 {모델명} ({자산번호}) 반납 예정일이에요!
[반납하기] [연장하기] 버튼
```
